Looping Constructs in C and C++

Loops are fundamental constructs in programming that allow for repetitive execution of a block of code. They are especially useful when paired with certain operators.

The while Loop

The while loop continues to execute a block of code as long as the condition remains true.

Example:
int i = 1;
while (i <= 10) {
printf("%d ", i);
i++;
}

Further Reading: while Loop in C

The for Loop

The for loop allows for initialization, condition checking, and incrementation in one line, making it compact for iterations.

Example:
for (int i = 0; i < 10; i++) {
printf("%d ", i);
}

Further Reading: for Loop in C

The do-while Loop

The do-while loop guarantees that the code block will be executed at least once before the condition is checked.

Example:
int i = 1;
do {
printf("%d ", i);
i++;
} while (i <= 10);

Further Reading: do-while Loop in C

Break and Continue Statements

break exits the loop immediately, while continue skips the rest of the loop and starts the next iteration.

Nested Loops

Nested loops are loops inside another loop. They are often used for multi-dimensional data structures.

Converting for loop to while loop

You can often convert a for loop into a while loop, although the for loop syntax is generally preferred for clarity when iterating a known number of times.

Infinite Loops

Infinite loops run indefinitely because their condition is always true and they lack an exit mechanism. Use with caution.

Example:
while (1) { /* code */ }

Running Sum

Keeping a running sum involves incrementally adding numbers to a total.

Example:
int sum = 0;
for (int i = 1; i <= 10; i++) {
sum += i;
}
printf("Sum = %d", sum);

System Calls

System calls such as printf() can be used within loops for output.
