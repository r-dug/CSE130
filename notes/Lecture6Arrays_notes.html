Introduction to Arrays

Arrays allow us to manipulate large groups of data of the same type. An array is a collection of elements where each element has a type and can be accessed using an index, which starts at zero.

  • Declaration: [];
  • Access: Elements are accessed using an index: [].
  • Initialization: Arrays can be initialized with comma-separated values inside curly braces: {value1, value2, ...}.
  • Out of Bounds: Accessing beyond the size of the array can lead to undefined behavior and can crash the program.

Example:

int scores[10]; // Declaration of an array of size 10
scores[0] = 90; // Access and assignment to the first element

Initializing Arrays

You can initialize arrays in several ways:

  • Provide a list of values explicitly.
  • Use a default value for all elements.
  • Omit the size of the array and let the compiler infer it from the initialization list.

Example:

int numbers[5] = {1, 2, 3, 4, 5}; // Explicit values
int zeros[5] = {0}; // All elements set to 0
int inferSize[] = {1, 2, 3}; // Size inferred as 3

String in C

In C, strings are NOT a datatype; they are represented as arrays of characters with a null terminating character at the end. This termination character is \0 and indicates the end of the string. When declaring a string, one extra element is required to accommodate this null terminator.

char name[5] = "John";  // John\0 - 'name' is 5 characters including the null terminator

Loop Initialization of Arrays

Loops can be used for initializing or processing arrays:

  • Using for loop to initialize values in the array.
  • Using for loop to print values in the array.

Example:

// This section of C code initialises an array, and an initial integer value of 10.
// Then the loop iterates over every index of the array, 
//     assigns the current value of the "value" variable and decrements its value,
//     then finally increments the index value until it is 10.
int value_array[10];
int value = 10;
for(int i = 0; i < 10; i++) {
    value_array[i] = value; // Initialize with values 10 to 1
    value --;
}

Multi-Dimensional Arrays

Arrays can have more than one dimension, like a 2D array (matrix), which uses two indices. Consider what you might use a matrix for in a C program.

  • Declaration: [][];
  • Access: Access elements using two indices: [][].
  • Initializing 2D Arrays: Values can be assigned using nested curly braces. You don't HAVE TO declare the size of the first dimension, but You probably should...

Example:

int matrix[2][3] = {1, 2, 3, 4, 5, 6}; // A 2x3 matrix
int matrix_2[2][3] =    {1, 2, 3, 
                        4, 5, 6}; // same matrix, but formatted for "human consumption"
int matrix_3[][3] =    {1, 2, 3, 4, 5, 6}; // the same matrix, but compiler will determine the first dimension

Processing Multi-Dimensional Arrays

Multi-dimensional arrays can be processed in a few ways...

  • process entire array
  • process a particular row
  • process a particular column
  • When processing a single row or column, we use algorithms similar to that of a one dimensional array
  • When processing a multi-dimensional array entirely, we likely use nested looping.
  • Arrays can come in more dimensions than two. It starts to become difficult to conceptualize. Think of them as collections of collections... This is often used in machine learning, graphics, etc.

Passing Arrays to Functions

Arrays can be passed to functions. The size of the array need not be specified when passing but must be known in the function to avoid out-of-bounds access.

Example:

// How would you initialize a "string" in a function? Try to do that...
void printArray(int array[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", array[i]);
    }
    printf("\n");
}

Handling Multi-Dimensional Arrays

When passing multi-dimensional arrays to functions, all dimensions except the first must be specified.

  • Passed by reference only
  • No check if array indices are in bounds. Be careful.
