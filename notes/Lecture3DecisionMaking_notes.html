Concept of Decision Making

Relational and Logical Operators

In C and C++, relational operators are utilized to compare values. These include:

  • > (greater than)
  • < (less than)
  • >= (greater than or equal to)
  • <= (less than or equal to)
  • == (equal to)
  • != (not equal to)

Logical operators are used to combine or negate boolean conditions:

  • ! (not)
  • && (and)
  • || (or)

If Statements

The if statement allows execution of a block of code if a condition is true.

Example syntax:

if (condition) {
    // code to execute if condition is true
}

Else and Else If Statements

Use else to provide an alternative execution path when the if condition is false.

The else if allows testing for multiple conditions in sequence.

Precedence of Operators

Understanding the rules of precedence is essential when evaluating complex expressions.

Switch Statements

The switch statement simplifies code when there are many possible conditions to check.

Ternary Operator

The ternary (condition ? expression1 : expression2) operator provides a shorthand for if-else.

Common Mistakes

Mixing up assignment = and equality == is a common mistake. Use == for comparison.

Random Numbers
