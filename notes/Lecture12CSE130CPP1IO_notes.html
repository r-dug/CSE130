Lecture: Introduction to C++ Programming

Similarities Between C and C++

After completing the C language sequence, students are now moving on to study their second language, C++. Due to the close relationship between the two languages, transitioning to C++ should be easier as many concepts are the same. You've already covered a large portion of the language in C, so the focus will be on what's new in C++. C++ is an object-oriented programming language, which will be a key focus of this course.

History of C++

C++ has its roots in C, with its variants and the progression from the traditional C language and its predecessor, language B. Invented in 1979 by Bjarne Stroustrup, C++ is now 44 years old. It is essential to understand this background to appreciate the evolution and design of the language.

Hello World Program in C++

A simple "Hello World" program in C++ includes familiar components such as library inclusions, main function, and an output statement using 'cout'. The introduction of new features like namespaces distinguishes C++ from C, which you will learn more about. Namespaces help in organizing code and avoiding naming conflicts.

Code:


#include <iostream>
using namespace std; // declares the namespace used by default

int main() {
    // std::cout 
    cout << "Hello World" << endl;
    return 0;
}

Input and Output in C++

In C++, input and output operations are done using streams, which are objects that handle data input and output. The 'cin' object is used for reading data in, and the 'cout' object is used for writing data out. These objects are part of the standard namespace and are included from the IO stream library.

Code:


#include <iostream> // necessary for input streams
using namespace std; // declares the namespace used by default

int main() {
    // std::cout would be used if namespace is not defined
    cout << "Welcome to C++" << endl;
    return 0;
}

Reading Data Using Extraction Operators

Extraction operators ( > > and < <) are used in C++ to read and write data streams. The direction of the operator indicates the flow of data, with 'cin' for input and 'cout' for output. Data types are crucial when using extraction operators to ensure correct data conversion.

Working with Strings

Strings in C++ are objects defined in the string library. You can easily output and input strings using 'cout' and 'cin' along with extraction and insertion operators. Remember to include the string library for string manipulation functions.

Outputting a String Example:


#include <string>
using namespace std;

int main() {
    string example = "Today is Monday";
    cout  < < example  < < endl; // the " < <" is an extraction operator
    return 0;
}

Inputting and Outputting Strings Example:

    
#include <iostream> 
#include <string>
using namespace std;

int main() {
    string name;
    cout << "What is your name? ";
    cin >> name; // we can save to a string object in c++, but needed to import the string header file.
    cout << "Hello, " << name << endl;
    return 0;
}
