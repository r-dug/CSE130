Conditional Statements

Making decisions based on conditions is fundamental in programming. We use relational and logical operators to evaluate conditions, which return boolean results (true or false, represented by non-zero and zero, respectively).

Relational Operators

    >   -- Greater than
    <   -- Less than
    >=  -- Greater than or equal to
    <=  -- Less than or equal to
    ==  -- Equality comparison (THIS WILL BE ON THE EXAM: Remember, it's double equals for equality checks)
    !=  -- Not equal

For further reading on relational operators: C++ operator comparison.

Logical Operators

    !   -- NOT
    && -- AND
    ||  -- OR

Truth tables can be useful to predict the outcome of logical expressions.

For further reading on logical operators: C++ logical operators.

If Statements

The if statement evaluates a boolean condition. If the condition is true, the following block of code, optionally enclosed by curly brackets, gets executed. 

    if (condition) {
        // executed if condition is true
    }

Else and Else If Clauses

In addition to if, you can specify alternative paths with else and else if: 

    if (condition1) {
        // executed if condition1 is true
    } else if (condition2) {
        // executed if condition1 is false and condition2 is true
    } else {
        // executed if condition1 and condition2 are false
    }

Switch Statement

A switch statement is another way to handle multiple possible conditions by comparing a variable against a series of values. 

    switch (expression) {
        case value1:
            // executed if expression == value1
            break;
        case value2:
            // executed if expression == value2
            break;
        // ...
        default:
            // executed if no cases match
    }

For further reading on switch statements: C switch statement.

Conditional (Ternary) Operator

The conditional operator is a compact way of writing simple if-else statements. 

    result = (condition) ? value1 : value2;

Mistakes to Avoid

Common Error: Confusing the assignment operator (=) with the equality operator (==). Be vigilant as the compiler may not always catch this.

Generating Random Numbers

Random numbers are generated in C using the rand() function. To get different sequences each run, seed the random number generator using srand() with a variable seed like the current time using time(NULL). 

    #include <stdlib.h>
    #include <time.h>
    
    srand(time(NULL));
    int randomNumber = rand() % upperLimit; // random number in [0, upperLimit-1]
