Lecture 8: Strings in C/C++

Introduction

Understanding strings in C/C++ is essential as they are commonly used for manipulating large chunks of text in numerous applications such as word processors and web browsers. Strings are not a built-in data type in C as integers or floats. They must be managed explicitly.

Essential Concepts

Decoding Strings

Strings in C/C++ are sequences of characters enclosed in double quotes (""""), terminating with a null character ("\0").

char name[] = "Alice"; // 'Alice\0'

THIS WILL BE ON THE EXAM: Understand the difference between a character and a string.

  • A character: 'A'
  • A string: "A" (which is 'A\0')

Characters can be represented by their ASCII values and can include letters, digits, and special characters.

String Declaration and Manipulation

Arrays of characters function as strings. Use the array or pointer notation for accessing and manipulating strings.

char city[7] = {'C', 'h', 'i', 'c', 'a', 'g', 'o'}; // 'Chicago\0'
char *pCity = "Chicago"; // Also points to the string 'Chicago\0'

NOTE: Arrays can store strings shorter than the size declared, with unused elements following the null-terminating character.

Input and Output

To read strings, scanf() can be used for string literals but stops at whitespace.

char color[12];
printf("Enter your favorite color: ");
scanf("%s", color);

NOTE: Strings can also be read entirely, including white spaces, using gets() function.

Key String Functions

Understanding the string handling functions strlen() (string length), strcmp() (string compare), and strcpy() (string copy) is crucial. These will be used frequently.

size_t len = strlen(name); // Returns length of 'name' excluding '\0'
int cmp = strcmp(name1, name2); // Compares name1 with name2
strcpy(dest, src); // Copies src into dest

THIS WILL BE ON THE EXAM: Understand how these string functions work.

String Comparison

Use strcmp() for comparing strings, which is necessary since strings cannot be compared with standard comparison operators like integers.

int compare = strcmp("A", "B"); // Compares 'A' with 'B'

Complex String Manipulation

Plentiful library functions exist for string manipulation, such as strcat(), strstr(), and strtok(). These can handle tasks such as:

  • Converting strings to numerical types for calculations
  • Finding and copying substrings
  • Tokenizing strings
