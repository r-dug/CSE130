Lecture 9: Data Structures in C/C++

Introduction to Structures in C

C allows the creation of compound data types using structures. This is particularly useful for combining multiple data types under a single name. A structure in C can be defined using the struct keyword.

Syntax:

struct StructName {
    DataType1 member1;
    DataType2 member2;
    // ... more members
} instanceName;

Defining a Structure

To define a structure without allocating memory:

struct MyExample {
    int number;
    char symbol;
    char str[100];
};

Creating a Structure Instance

Allocating memory for an instance:

struct MyStructure {
    // members...
} myStructInstance;

Example: Defining a Student Structure

The following structure represents a student with various fields:

struct Student {
    char firstName[50];
    char lastName[50];
    int courseGrades[10];
    float gpas[10];
    // ...
};
  • You should understand how to define and use structures in C.

Accessing Structure Members

Members can be accessed using the . (dot) operator:

Student newStudent;
newStudent.gpa = 3.85; // Set the GPA

Structure Assignment and Comparison

  • Structures support the assignment operator.
  • Structures do not have a built-in comparison operator; it must be defined based on specific member comparison.
  • Custom functions are needed for printing and input operations of structures.

Typedef Alias and Pointers

Creating an alias for a type:

typedef int Length;
Length l = 20;

Using typedef with Structures:

typedef struct Student StudentAlias;
StudentAlias myStudent;

Accessing members using pointers:

Student *ptr = &newStudent;
ptr->gpa = 3.85; // Same as newStudent.gpa = 3.85;

Comparison of Arrays and Structures

  • Structures can be passed by value or by reference, whereas arrays are passed by reference only.
  • Differences in assignment and size handling.
  • Aggregate operations:
    • Arrithmetic - Array: no | Struct: no
    • Assignment - Array: no | Struct: Yes
    • I/O - Array - Strings only | Struct: no
    • Comparison - Array: no | Struct: no (but elements can be compared)
    • Parameter passing - Array: by reference | Struct: by value OR reference
    • Function refurning a value - Array: no | Struct: Yes

list struct

A union allows different members to occupy the same memory space; only one member is active at a time. Useful for saving memory when the union members are mutually exclusive.

Example of a Union:

int array_size = 100;
struct list 
{
    int elements [array_size];
    int length;
};

Unions

A union allows different members to occupy the same memory space; only one member is active at a time. Useful for saving memory when the union members are mutually exclusive.

Example of a Union:

union Value {
    float fNumber;
    int iNumber;
    char cCharacter;
};

Type Casting

Type casting in C allows conversion between compatible types. Syntax for type casting: (Type)variable.

Example with Integers and Floats:

int x = 10, y = 20;
float result = (float)x / (float)y;
