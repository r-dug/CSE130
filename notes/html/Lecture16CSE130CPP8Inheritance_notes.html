Lecture Notes on Inheritance in C++:

Inheritance in C++ allows for the creation of customized reusable code by deriving new classes from existing ones. It enables the creation of more specialized classes with additional features while retaining the properties of the base class. There are two main types of inheritance: single inheritance (deriving from one base class) and multiple inheritance (deriving from multiple base classes).

Creating Inheritance Hierarchy:

  • Inheritance involves passing attributes from existing classes into new ones, starting from a base class to more derived classes.
    • "is-a" relationship
    • EX: "every 'employee' is-a 'person'"
  • Syntax for creating an inherited class: 
    class DerivedClass : accessSpecifier BaseClass // accessSpecifier is public, protected or private
{
    member list
};
  • Access specifiers (public, protected, private) determine the visibility of the base class members in the derived class.
  • Single inheritance: derived class has a single base class
  • Multiple inheritance: derived class has multiple base classes

Example - single inheritance:

class Shape {
public:
    // Base class members
};

class Circle : public Shape {
public:
    // Custom members for Circle
};

Inheritance Visualization:

  • Inheritance can be visualized as a tree- building on preexisting functionality, where each derived class (leaf/branch) gains additional capabilities from the base classes(parent stem/branch).
  • Inheritance enables the creation of specialized classes without rewriting existing functionality.

Overriding Functions:

  • In derived classes, functions can be overridden to provide specific implementations.
  • Redefining a function in a derived class with the same signature as the base class function results in overriding behavior.
  • Use of the scope resolution operator (::) to specify which class's function is being called.

Example of function overriding:

class Mammal {
    public:
        string name;
        void makeSound() {
            cout << name << "Ooga booga!" << endl;
        };
    };
    
class Dog : public Mammal {
    public:
        void makeSound() {
            std::cout << "Woof" << std::endl;
        }
};

Virtual Functions and Abstract Classes:

  • Virtual functions in C++ allow for dynamic function binding based on object type.
  • Use of the virtual keyword indicates that the function can be overridden in derived classes.
  • Abstract classes are classes that sre too vague to actually be instantiated but still have properties.
  • Pure virtual functions are declared with virtual and set to 0, forcing derived classes to implement the function.

Example of Abstract Class:

class Mammal {
public:
    string name;
    virtual void makeSound() = 0; // Pure virtual function
};

class Dog : public Mammal {
public:
    void makeSound() {
        cout << "Woof" << endl;
    }
};

Inheritance Chain and Constructors:

  • Constructors in derived classes should call the corresponding base class constructors explicitly to ensure proper initialization.
  • Constructor and destructor chaining occurs in the opposite order while creating and destroying objects in an inheritance hierarchy.

Example of incorrectly implemented access:

class Figure {
protected:
    int x, y;
};

class Circle : public Figure {
public:
    int radius;
};

int main() {
    Circle a; // create circle object 'a'
    a.x = 0; // illegal. cannot access protected member from main
    a.y = 0; // illegal. cannot access protected member from main
    a.radius = 10; //legal - can access public member 
}

Example of correctly implemented access:

class Figure {
protected:
    int x_, y_;
};

class Circle : public Figure {
    private:
        int radius_;
    public:
        Circle(int x, int y, int radius)
};
//constructor for Circle
Circle::Circle(int x, int y, int radius) {
    x_ = x; // this assignment can be done because of the protected access level in 'Figure' objects. an alternative would be to create accessor functions were the access level private...
    y_ = y;
    radius_ = radius;
}
int main() {
    Circle a(0,0,10); // create circle object 'a'
}
