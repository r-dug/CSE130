Boolean Data Type

In C++, the Boolean data type (bool) holds true (1) or false (0) as its value. True and false are reserved keywords. This data type is used for conditional checking in programs.


bool status = false;
if (status) {
    // First statement executed
} else {
    // Else condition executed
}

Function Overloading

Function overloading allows multiple functions with the same name but different parameters or parameter types. This enables the same function name to have different implementations based on the arguments provided.


// Overloaded function for swapping values of different data types
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}
void swap(float *a, float *b) {
    float temp = *a;
    *a = *b;
    *b = temp;
}
void swap(double *a, double *b) {
    double temp = *a;
    *a = *b;
    *b = temp;
}
void swap(char *a, char *b) {
    char temp = *a;
    *a = *b;
    *b = temp;
}
int main(void){
    int a = 10;
    int b = 20;
    float c = 10.1, d = 11.2;
    char x = 'x', y = 'y';
    swap(&a, &b);
    swap(&c, &d);
    swap(&x, &y);
    cout << a << " " << b << endl;
    cout << c << " " << d << endl;
    cout << x << " " << y << endl;
    return 0;
}

Default Arguments

Default arguments allow parameters with default values. These parameters can be overridden by providing specific values during function calls.


// Function with default arguments for computing volume of a box
int boxVolume(int length = 1, int width = 1, int height = 1) {
    return length * width * height;
}
boxVolume(); // Uses default values
boxVolume(10, 5, 10); // Explicitly provided values
boxVolume(10, 10); //  ambiguously provided values

Scope Resolution Operator and Static Variables

The scope resolution operator (::) allows specifying the scope of a variable or object. Static variables retain their value between function calls, providing continuity and persistence (described as functions with memory in the lecture).


// Scope resolution operator example
int count = 0;
int main() {
    int count = 0;
    ::count = 5; // Accessing global variable
    count = 2; // Accessing local variable 
}

// Static variable example
int counter() {
    static int x = 0;
    x++;
    return x;
}
